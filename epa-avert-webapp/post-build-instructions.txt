--------------------------------------------------------------------------------
   Post-Build Instructions for Deploying Updates to Cloud.gov:
--------------------------------------------------------------------------------

1) Edit 'index.html':
   Move details-polyfill.js script to be last element inside HTML body

2) Copy and replace entire 'build' directory contents into 'public'
   directory in 'epa-avert-webservice/app' directory

3) Redeploy updated 'epa-avert-webservice' app to Cloud.gov:
   cf push
